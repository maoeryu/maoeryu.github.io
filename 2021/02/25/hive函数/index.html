关系运算符

运算符之间的比较是否成立来生成true或false值.

运算符 数据类型 描述
a = b 所有原始类型 如果表达式a等于表达式b,则为true,否则为false
a == b 所有原始类型 同上
a <=> b 所有原始类型 如果都非null,则同=,如果都为null,则为true,如果其一为null,则为false
a <> b 所有原始类型 如果其一为null,则为null,如果a不等于b,则为true,否则为false
a != b 所有原始类型 同上
a < b 所有原始类型 同上,小于
a <= b 所有原始类型 同上,小于等于
a > b 所有原始类型 同上,大于
a >= b 所有原始类型 同上,大于等于
a [not] between b and c 所有原始类型 同上,a>=b,a<=c,not取相反结果
a is null 所有类型 如果a为null,则为true,否则为false
a is not null 所有类型 同上,相反结果
a is [not] (true|false) 布尔类型 仅当a满足条件时为true
a [not] like b string 如果其一为null,则为null,如果a匹配正则b,则为true
a rlike b string 同上,b为java正则
a regexp b string 同rlike

算术运算符

各种常见算术运算.返回number类型.
如果任何操作数为null,则结果也为null.

运算符 数据类型 描述
a + b number 相加,返回父级类型,如float+int,返回float
a - b number 同上,相减
a * b number 同上,相乘,如溢出时,则必须将其中一个转换成父级类型
a / b number 同上,相除
a % b number 同上,返回相除的余数
a div b integer 返回相除的倍数
a & b number 按位与,均为1时才为1
a b number
a ^ b number 按位异或,仅有一个为1时为1
~a number 按位取反

逻辑运算符

返回布尔值true/false/null.null表现为"unknown"标志.

运算符 数据类型 描述
a and b boolean 都为true时,为true,否则为false,如其一为null,则为null
a or b boolean 如其一为true,则为true,否则为false,false和null时为null,其它为false
not a boolean a为false时为true,为null时为null,否则为false
! a boolean 同上
a in (val1,val2,...) boolean 如a等于任一值,则为true
a not in (val1,val2,...) boolean 如a不等于任一值,则为true
[not] exists (subquery) 如果子查询返回至少一行,则为true

聚合函数(udaf)

接受从0行到多行的0个到多个列,然后返回单一值.

数据类型 函数 描述
bigint count(*) 包含null的行数
bigint count(expr) 非null
bigint count(distinct expr[,expr...]) 唯一且非null
double sum(col),sum(distinct col)
double avg(col),avg(distinct col)
double min(col)
double max(col)
array collect_set(col)
array collect_list(col)
integer ntile(integer x)

聚合操作时要注意null值.
count(*) 包含null值,统计所有行数.
count(id) 不包含null值.
min 求最小值是不包含null,除非所有值都是null.
avg 求平均值也是不包含null.
