centos安装

1)卸载系统自带的Mariadb
查询出已安装的mariadb,
rpm -qa|grep mariadb

卸载文件名为上命令查出的所有文件,
rpm -e --nodeps 文件名

2)删除etc目录下的my.cnf文件
rm /etc/my.cnf

3)执行以下命令来创建mysql用户组,并加入组

4)将下载的二进制压缩包放到/usr/local/目录下
5)解压安装包,
tar xzf /home/centos/mysql-5.5.57-linux-glibc2.12-x86_64.tar.gz

6)将解压好的文件夹重命名为mysql
7)在etc下新建配置文件my.cnf,并在该文件内添加以下代码,

8)创建目录并将其用户设置为mysql
yum -y install numactl.x86_64

将mysql的bin目录加入PATH环境变量,编辑/etc/profile文件,

以root账户登陆mysql,默认是没有密码,

9)重新登录
