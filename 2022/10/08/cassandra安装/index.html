cassandra安装

Posted on 2022-10-08 In 数据库

集群部署

下载安装

安装jdk,配置好环境变量.
下载并解压安装包.

创建文件夹

创建data/commitlog/saved_caches文件夹.

修改配置文件

修改conf下的cassandra.yaml文件.

cluster_name:集群名称
data_file_directories:数据目录
commitlog_directory:日志目录
saved_caches_directory:缓存目录
seeds:集群种子节点ip,新加入的节点从种子节点同步数据,可为多个,中间逗号分隔

修改完分发各节点.

各节点分别配置cassandra.yaml的listen_address和rpc_address参数.
listen_address:监听地址(本机ip或主机名)
rpc_address:rpc地址(本机ip或0.0.0.0)
两个配置参数分别修改为当前机器的ip.

配置环境变量

export CASSANDRA_HOME=/opt/cassandra
export PATH=$PATH:$CASSANDRA_HOME/bin

启动

bin/cassandra -p cassandra.pid -R

查看集群状态

bin/nodetool status

节点停止

删除节点

nodetool removenode [Host ID]

集群扩容(横向扩容)

将任意节点的安装包拷贝到新节点下.
清空data/commitlog/saved_caches目录下的内容.
修改cassandra.yaml,将listen_address和rpc_address参数设为本机地址.
配置环境变量.
启动节点.

此时节点会自动加入到集群中.
查看数据迁移情况/集群状态.
bin/nodetool netstats

备份和恢复

创建快照

1.创建单个空间的快照
nodetool snapshot -t [快照名称] [空间名称]

2.创建所有空间的快照
nodetool snapshot -t [快照名称]

-t后面指定快照的名字,备份好的数据在这个目录下
$CASSANDRA_HOME/data/yourkeyspace/table-uuid/snapshots/20220512

例如:
nodetool snapshot -t 20220512 metled
$CASSANDRA_HOME/data/test/user-1e68c6a0d19811ec8b0991145c5c7379/snapshots/20220512

快照查询

1.查找快照目录
find -name snapshots

2.列出快照
nodetool listsnapshots

3.清除快照
nodetool clearsnapshot -t [快照名称]
或者
nodetool clearsnapshot --all [空间名]

4.清空所有快照
nodetool clearsnapshot --all

5.恢复数据
导入某张表快照的数据,如果表不存在或者结构发生改变请运行schema.cql来创建表
nodetool import -- [空间名] [表名] [快照地址]

例如:
nodetool import -- catalogkeyspace journal \ $CASSANDRA_HOME/data/catalogkeyspace/journal-c4337610d1b711ec8b0991145c5c7379/snapshots/catalog-ks

快照地址可以通过以下命令查看
shell find -name [快照名称]

数据迁移

假设需要将表数据迁移到新的集群上,则需要在旧集群的每个节点上执行.
xsstableloader --nodes [新集群ip] [表的快照地址]

例如:
shell sstableloader --nodes 192.168.160.103 \ $CASSANDRA_HOME/data/test/user-1e68c6a0d19811ec8b0991145c5c7379/snapshots/222

nodetool工具

查看集群运行状态
nodetool status

参看某个节点负载
nodetool info

查看各个CF的详细统计信息,包括读写次数/响应时间/memtable信息等.
nodetool cfstats

基本操作

连接cqlsh
bin/cqlsh hadoop-sh1-core1

键空间操作

查询键空间
DESC KEYSPACES;

创建键空间
CREATE KEYSPACE test WITH replication = {'class':'SimpleStrategy', 'replication_factor' : 2};

删除键空间
drop keyspace test;

选择键空间
USE test;

表操作

创建表

查看所有表
desc tables;

查看表结构
desc table [表名]

清空表
truncate [表名]

删除表
DROP TABLE user

添加字段
ALTER TABLE [表名] ADD [字段名称] [字段类型];
例如
ALTER TABLE student ADD phone text;

删除字段
ALTER table [表名] DROP [字段名称];
例如:
ALTER TABLE student DROP student_email;

插入表
INSERT INTO [keyspace_name.] table_name (column_list) VALUES (column_values) ;
