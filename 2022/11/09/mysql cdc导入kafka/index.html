使用Debezium/flink sql,将mysql数据实时导入kafka.

https://debezium.io/documentation/reference/stable/connectors/mysql.html#mysql-connector-properties
https://ververica.github.io/flink-cdc-connectors/master/content/about.html

flink-sql-connector-mysql-cdc-2.2.0.jar
flink-sql-connector-kafka-1.15.2.jar
mysql开启binlog日志.

使用flink stream api

关联kafka

kafka里面的flink-cdc-kafka这个主题会被自动创建.
如果想指定一些属性,可以提前手动创建好主题.

建立同步任务

可以退出flink sql-client.
进入flink web-ui,可以看到mysql表数据已经同步到kafka中了,对mysql进行插入,kafka都会同步更新.

mysql再次执行以下sql后,查看kafka消息.

使用debezium

https://debezium.io/releases/1.5/

低版本的kafka暂不支持,本次使用kafka_2.11-2.1.0
debezium 1.6及以上需要java 11.

下载debezium-connector-mysql-1.5.4.Final-plugin.tar.gz.将解压后的jar包全部加入kafka lib目录下.
debezium-connector-mysql-1.5.4.Final.jar

antlr4-runtime-4.7.2.jar
