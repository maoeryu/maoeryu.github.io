elasticsearch-连接查询

Posted on 2021-11-24 Edited on 2022-07-26 In 数据库

在像Elasticsearch这样的分布式系统中执行完整的SQL风格的连接是非常昂贵的,相反Elasticsearch提供了两种设计为水平伸缩的join形式.

nested(嵌套查询)

文档可以包含类型为nested的字段,这些字段用于索引对象数组,其中每个对象都可以查询(使用嵌套查询)作为独立文档.

has_child/has_parent

一个join字段关系可以存在于单个索引中的文档之间,has_child查询返回子文档与指定查询匹配的父文档,has_parent查询返回其父文档与指定查询匹配的子文档.

需要配置search.allow_expensive_queries=true.

elastic版本7.10.0.

elasticsearch-book

Posted on 2021-11-23 Edited on 2022-07-26 In 数据库

在Elasticsearch中,文档归属于一种类型(type),而这些类型存在于索引(index)中.

Relational DB -> Databases -> Tables -> Rows -> Columns
Elasticsearch -> Indices -> Types -> Documents -> Fields

Elasticsearch集群可以包含多个索引(indices)(数据库),每个索引可以包含多个类型(types)(表),每个类型包含多个文档(documents)(行),每个文档包含多个字段(Fields)(列).

java8 Stream

Posted on 2021-09-24 Edited on 2022-07-26 In 编程语言

Stream是Java8中处理集合的关键抽象概念,它可以指定你希望对集合进行的操作,可以执行非常复杂的查找、过滤和映射数据等操作.
使用StreamAPI对集合数据进行操作,就类似于使用SQL执行的数据库查询.
也可以使用StreamAPI来并行执行操作.
